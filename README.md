# 7x7 Health Tic-Tac-Toe game

## Why?
I built this game with a purpose to test out Lovable. It was part of [AI Prototyping course by GoPractice]([url](https://gopractice.io/course/aiprototyping/)). 

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Simply open [Lovable](https://lovable.dev/projects/a8e9129a-fb23-4b6b-ae62-e54acf88c093) and click on Share -> Publish.

## Can I connect a custom domain to my Lovable project?

Yes, you can!

To connect a domain, navigate to Project > Settings > Domains and click Connect Domain.

Read more here: [Setting up a custom domain](https://docs.lovable.dev/features/custom-domain#custom-domain)
